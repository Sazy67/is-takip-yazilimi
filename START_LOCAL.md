# Local'de Çalıştırma Talimatları

## 1. Backend'i Başlat

Terminal 1'de:
```bash
cd backend
python app.py
```

Backend http://localhost:5000 adresinde çalışacak.

## 2. Frontend'i Başlat

Terminal 2'de:
```bash
cd frontend
npm run dev
```

Frontend http://localhost:3000 adresinde çalışacak.

## 3. Tarayıcıda Aç

http://localhost:3000 adresine git.

## Kullanıcı Bilgileri

- Admin: `admin` / `admin123`
- User: `user` / `user123`

## Veritabanı

Veriler `backend/kayitlar.db` dosyasında saklanıyor. Bu dosya silinmediği sürece veriler kalıcı.
