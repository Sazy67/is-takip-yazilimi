# İş Takip Yazılımı

Modern ve kullanıcı dostu iş takip uygulaması.

## Özellikler

- 🔐 Kullanıcı girişi (Admin/User rolleri)
- 📝 Kayıt ekleme, düzenleme, silme
- 🔍 Arama ve filtreleme
- 📱 Responsive tasarım
- 🎨 Modern gradient UI

## Teknolojiler

- **Frontend**: React + Vite
- **Backend**: Flask (Python)
- **Database**: SQLite
- **Styling**: CSS3

## Hızlı Başlangıç

### Gereksinimler
- Python 3.8+
- Node.js 16+

### Kurulum

1. **Backend**
```bash
cd backend
pip install -r requirements.txt
python app.py
```

2. **Frontend** (Development)
```bash
cd frontend
npm install
npm run dev
```

### Production Build

```bash
cd frontend
npm run build
cd ../backend
python app.py
```

Uygulama: http://localhost:5000

## Kullanıcı Bilgileri

- **Admin**: `admin` / `admin123`
- **User**: `user` / `user123`
